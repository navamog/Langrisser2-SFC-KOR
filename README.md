# Langrisser2-SFC-KOR
 데어랑그릿사 한글패치
 (Der Langrisser - Korean Translation Patch)

■ 소개
  슈퍼패미컴(SFC/SNES)용 「데어랑그릿사(デア・ラングリッサー)」의
  일본어 → 한국어 번역 패치입니다.
  byuu(near) 님의 Der Langrisser 영문 번역 툴킷을 기반으로 제작되었습니다.

■ 적용 방법
  1) 원본 ROM 준비 [해시 확인 필수]
     - 「Der Langrisser (Japan)」 일본판 원본 ROM
     - 크기 : 2,097,152 바이트 (2MB / 16Mbit), 헤더 없음(headerless)
     - MD5  : 91d62c4cb790fc2fb38b10b68616e228
     - CRC32: 35F9EECC
     - SHA1 : E2CB4935CFB5B442928ECAD43DA8B59EE5B159E4
     ※ 반드시 위 해시와 일치하는 ROM에 적용하세요. 해시가 다르면 패치가
       적용되지 않거나 정상 동작하지 않습니다.
       (헤더가 붙은 2,097,664바이트 ROM이면 헤더를 먼저 제거하세요.)

  2) IPS 패치 적용
     - 원본 ROM에 「DerLangrisser_Korean_v1.0.ips」를 적용
     - 패치 도구 : Lunar IPS(Flips), NINJA, beat 등
     - 적용 후 ROM 크기 : 4MB (한글 폰트/스크립트 영역만큼 확장됩니다)

■ 라이선스 / 주의
  - 개인 학습·연구 목적의 패치입니다. 원본 ROM은 정당하게 소유한 것만 사용하세요.
  - 본 배포물에는 IPS 패치 파일만 포함됩니다. 원본 ROM 또는 패치된 ROM은
    배포하지 않으며, 앞으로도 배포하지 않습니다.
  - 재배포 시 본 안내문(README)을 반드시 포함해 주세요.
  - 원칙적으로, 해당 기종의 원본 롬 카트리지를 보유한 분이 개인적으로만
    이용한다는 전제 하에 본 한글패치 파일을 다운로드할 수 있습니다.
  - 상기 조건에 해당하지 않는 분이 본 한글패치 파일을 다운로드하여 이용할 경우
    발생하는 모든 책임은 이용 당사자에게 있으며, 패치 제작자는 어떠한 책임도
    지지 않습니다.
  - 본 한글패치 파일 자체를 제작자의 허가 없이 타 사이트에 재배포하지 말아 주시고,
    재배포를 원하실 경우 패치 원본 게시물의 링크를 공유해 주시기 바랍니다.
    (아카이빙 포함)
  - 단, 제작자의 사망이 확인된 경우에는 예외로 합니다.
    (제3자 통보 후 3년 혹은 마지막 활동 기록 후 5년)
  - 모든 저작권은 게임 개발사·유통사(NCS / Masaya 등)에게 있으며, 패치 제작자는
    개발사·유통사에게 어떠한 권리도 주장하지 않습니다.
  - 본 패치는 byuu(near) 님의 Der Langrisser 영문 번역 툴킷(BSD License)을
    기반으로 제작되었습니다. 원저작 툴킷의 라이선스를 존중합니다.
